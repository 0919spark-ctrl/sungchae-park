Four Eyes XR Lab, UC Santa Barbara

XR User Study Generator for Haptic Interaction Research

Developed a Unity-based experimental framework in the Four Eyes XR Lab at UC Santa Barbara to generate counterbalanced XR user study trials using Balanced Latin Square counterbalancing and export structured JSON experiment configurations. Designed and implemented a complete XR interface system using the Meta XR SDK, enabling researchers to configure experiment parameters, generate trial sequences, and test setups interactively. Built Python validation scripts to verify generated JSON data and ensure balanced experimental conditions across participants Source code will be made publicly available after the associated research publication.

Role: Unity Developer / Research Engineer

Tech: Unity, C#, JSON, Meta SDK, Python

Code: Private (research project - will be released after publication)
